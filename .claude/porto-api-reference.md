## Usage Notes

Porto SDK is used for:
- Embedded wallet functionality
- Sign-In with Ethereum (SIWE) authentication
- Wallet abstraction layer
- Integration with viem for Ethereum interactions
- Customizable theming for wallet UI components
- Fee sponsoring and gasless transactions
- Payment processing and subscriptions
- Permission management for wallet operations
- Guest mode for onboarding without immediate account creation
